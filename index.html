Sobre o Divas 2000

O Divas 2000 é um projeto dedicado a celebrar as grandes artistas femininas da música pop que definiram a cultura e os hits dos anos 2000 até hoje.

Nosso objetivo é proporcionar aos fãs uma experiência nostálgica e ao mesmo tempo atual, trazendo os maiores sucessos dessas artistas icônicas que transformaram a indústria musical.

Este projeto foi desenvolvido utilizando:

  • HTML5 semântico e CSS moderno
  • JavaScript e manipulação avançada de DOM
  • Design responsivo com abordagem Mobile First
  • Framework Bootstrap 5
  • Arquitetura Single Page Application (SPA)
  • Integração com API do YouTube via Axios
